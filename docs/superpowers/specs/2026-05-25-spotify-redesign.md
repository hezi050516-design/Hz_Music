# Spotify 风格 UI 重设计

## 概述

将音乐 App 全部页面的视觉风格重构为 Spotify Dark 风格，引入 Lucide 图标库，底部 Tab 导航替代顶部 NavBar，统一搜索体验。

## 技术变更

| 项目 | 内容 |
|------|------|
| 图标库 | 新增 `lucide-react` |
| 导航 | 顶部 NavBar → 底部三 Tab（音乐库/歌手/设置） |
| 搜索 | 分散搜索 → 顶部栏统一搜索，跨 Tab 展示结果 |
| 专辑/封面 | 全部移除，改用纯色首字头像 |

## 配色方案

| 用途 | 色值 |
|------|------|
| 页面主背景 | `#121212` |
| 卡片/列表项 | `#181818` |
| 悬停/交互态 | `#282828` |
| 主强调色 | `#1DB954` (绿色) |
| 主文字 | `#FFFFFF` |
| 辅助文字 | `#B3B3B3` |
| 弱化文字 | `#727272` |

## 全局布局

```
┌───────────────────────────┐
│  顶部栏 (用户名 + 标题 + 搜索) │  ← 48px
├───────────────────────────┤
│      内容区 (scroll)        │
├───────────────────────────┤
│  播放条 (歌名 + 进度 + 控制)   │  ← 64px
│  音乐库 │ 歌手 │ 设置          │  ← 56px Tab 栏
└───────────────────────────┘
```

- 顶部栏：用户名 + 当前页标题 + Lucide Search 图标
- 点击搜索 → 全宽搜索栏，搜索结果实时过滤（歌曲 + 歌手）
- 播放条固定在 Tab 栏上方
- 三 Tab 底部导航替代原 NavBar

## 各页面设计

### 音乐库

- 顶部收藏卡片（歌曲总数快捷入口）
- 歌曲列表：每行左侧随机纯色方块（首字）+ 标题/歌手 + 时长
- 选中行绿色左边框
- 下方最近歌手快捷卡片区
- 登录入口放设置页

### 歌手页

- 2 列卡片网格，每张纯色背景 + 歌手名 + 歌曲数
- 无专辑封面，纯文字设计
- 点击进入歌手详情：顶部大色块歌手名 + 歌曲列表

### 详情页

- 全屏歌词沉浸，透明背景
- 左上角 Lucide ArrowLeft 返回按钮
- 歌名歌手名居中置顶
- 歌词居中滚动，当前行绿色 `#1DB954` 高亮
- 点击播放条也可返回

### 上传页

- 虚线边框大号上传区域，居中上传图标
- 选文件后弹出填写歌名/歌手对话框
- 未登录引导去设置页

### 设置页

- 服务器地址输入框 + 保存
- 账号信息 + 退出登录
- 播放模式快捷切换

### 播放条

- 绿色进度条 + 拖拽圆点
- 歌名/歌手左对齐
- 播放控制居中：模式切换 / 上一首 / 播放暂停 / 下一首 / 播放列表
- 音量滑块右侧
- 点击歌名区：进详情页 / 从详情页返回

## 组件变更清单
